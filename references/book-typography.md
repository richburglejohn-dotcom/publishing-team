# Culinary Crescendo — Book Interior Typography
### Confirmed: 2026-07-09, from Book 1's Amazon "Look Inside" preview

## Fonts
- **Chapter headings & drop caps:** Playfair Display
- **Body text:** EB Garamond
- **(Also used site-wide, third face reserved for accents/code-style elements):** Courier Prime

This matches the exact font stack already imported across the website (`about.html`, `blog.html`, `media.html`):
```
family=Playfair+Display:ital,wght@0,600;0,700;1,600&family=EB+Garamond:ital@0;1&family=Courier+Prime
```
Book 1's interior and the website share the same brand typography — not a coincidence, treat it as the established standard.

## Layout details (from Book 1's preview)
- Drop cap on the first letter of each chapter's opening paragraph, spanning roughly two lines, set in the heading face (Playfair Display)
- Chapter title format: "Chapter [N]: [Title]" — bold, centered, with a divider rule beneath
- Body paragraphs: classic first-line indentation, justified alignment
- Standard KDP trim formatting

## For the Prequel
Use this same pairing (Playfair Display for chapter headings/drop caps, EB Garamond for body) when formatting The Scarlet Foundation's interior, so it matches Book 1 and the website. This closes out the open typography question from earlier in the project.
